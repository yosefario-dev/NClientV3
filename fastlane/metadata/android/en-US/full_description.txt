A configurable manga client for Android.
Forked from NClientV2 by Dar9586, updated for modern Android.

Requires Android 7.0 (API 24) and above.

The app ships as an empty shell. Configure a supported source in Settings to unlock full functionality.

Features
 - Browse main page
 - Search by query or tags
 - Include or exclude tags
 - Blur or hide excluded tags
 - Download galleries
 - Favorite galleries
 - PIN lock for privacy
 - Share galleries
 - Open in browser
 - Bookmarks
 - History

What's new in V3
 - Material 3 / Material You design
 - Fixed for Android 15+ (edge-to-edge, status bar)
 - Fixed CDN webp image loading
 - Updated build system and dependencies
 - Removed crash reporting / tracking
 - App ships as SFW empty shell by default
