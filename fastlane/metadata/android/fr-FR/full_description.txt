Un client manga configurable pour Android.
Fork de NClientV2 par Dar9586, mis à jour pour Android moderne.

Nécessite Android 7.0 (API 24) et supérieur.

L'application est livrée comme une coquille vide. Configurez une source prise en charge dans les paramètres pour débloquer toutes les fonctionnalités.

Fonctionnalités
 - Parcourir la page principale
 - Recherche par requête ou par tags
 - Inclure ou exclure des tags
 - Flouter ou masquer les tags exclus
 - Télécharger des galeries
 - Galeries favorites
 - Verrouillage par PIN
 - Partager des galeries
 - Ouvrir dans le navigateur
 - Signets
 - Historique

Nouveautés dans V3
