Un client Android NHentai modernizzato e non ufficiale.
Fork di NClientV2 di Dar9586, aggiornato per Android moderno.

Richiede Android 7.0 (API 24) e superiore.

Funzionalità
 - Sfoglia la pagina principale
 - Cerca per query o per tag
 - Includi o escludi tag
 - Sfoca o nascondi i tag esclusi
 - Scarica gallerie
 - Gallerie preferite
 - Blocco con PIN
 - Condividi gallerie
 - Apri nel browser
 - Segnalibri
 - Cronologia

Novità nella V3
 - Design Material 3 / Material You
 - Correzione per Android 15+ (edge-to-edge, barra di stato)
 - Correzione del caricamento immagini CDN webp
 - Aggiornamento del sistema di build e delle dipendenze
 - Rimosso il report dei crash / tracciamento
